# ASR-0006

**Priority:** 2000
**Status:** Active
**Created:** 2026-04-10
**Updated:** 2026-04-10

## Requirement
All interface elements must remain interactive during any operation, and any operation that cannot complete near-instantly must display a progress indication for its duration.

Sourced from old ASR-003. Related decisions: ADR-022, ADR-023.
