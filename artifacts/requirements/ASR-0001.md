# ASR-0001

**Priority:** 0
**Status:** Active
**Created:** 2026-04-10
**Updated:** 2026-04-10

## Requirement
All credentials and tokens generated or stored by the system must use a cryptographically secure source of randomness and must never appear in any log, error message, API response, or source artifact.

Sourced from old ASR-007. Related decision: ADR-001.
